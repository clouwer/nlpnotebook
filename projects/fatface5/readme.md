## 小肥脸5号



一个聊天机器人的实现~~~~



### 拟实现功能

API部分

- 微信端口连接
- 图灵机器人API
- BosonNLP

数据获取部分

- 豆瓣小组爬虫 
- 微信聊天记录语料整理

聊天机器人部分

- NLU，NLG
- 人格信息
- 敏感词过滤
- 斗图



### 产品定位

- 任务驱动式的半闲聊机器人
- 主要工作是夸到开心
- 偶尔还能查查天气，测测运势
- 基于规则的小游戏模块



### 数据获取

使用什么数据集训练决定了对话类型

- 我们需要更多的爬虫
- 夸夸模块对话语料来源：
  - [豆瓣相互表扬小组](https://www.douban.com/group/kuakua/)
- 情侣聊天语料来源：
  - 与女朋友的聊天记录



### 意图识别

由于任务驱动 + 功能局限的特性，实现方案优先考虑基于规则的排序方案
